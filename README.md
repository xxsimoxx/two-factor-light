# Two Factor Light

This is a fork of [Two Factor](https://wordpress.org/plugins/two-factor/) v. 0.15.0 by WordPress.org that works in [ClassicPress](https://www.classicpress.net/).

It supports only Email and Backup codes, but can be extended to use WebAuthn using [WebAuthn Provider for Two Factor](https://wordpress.org/plugins/two-factor-provider-webauthn/) by Volodymyr Kolesnykov.

Use it at your own risk.
